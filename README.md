# SnatchingStore Shopify Theme

Marketplace-style Shopify theme starter for SnatchingStore.

## What is included

- Jumia-inspired marketplace homepage structure
- Search-first header
- Category sidebar and category tiles
- Featured products and flash deals shelves
- Collection grid
- Product detail page
- Footer with store and policy links

## Shopify connection

This repo is ready to be committed with GitHub Desktop and connected in Shopify through:

Online Store > Themes > Add theme > Connect from GitHub

Use the `main` branch once these files are committed and pushed.
